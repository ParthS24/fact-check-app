# AI Fact-Checking Web App

An AI-powered web application that automatically verifies factual claims from uploaded PDF documents using Large Language Models and live web search.

---

## Features

- Upload PDF documents
- Extract factual claims and statistics
- Perform live web verification
- Detect:
  - Verified claims
  - Inaccurate claims
  - False claims
- AI-generated fact-checking explanations
- Simple and interactive Streamlit interface

---

## Tech Stack

- Python
- Streamlit
- Gemini API
- DuckDuckGo Search
- pdfplumber

---

## Project Workflow

1. User uploads a PDF
2. Text is extracted from the PDF
3. AI extracts factual claims
4. Live web search gathers evidence
5. AI verifies the claims
6. Results are displayed with explanations
